# Java-Practices 📚

Roadmap completo de **Java para PAF1** - 7 temas progresivos con ejercicios paso a paso.

## 📖 Contenido

1. **01-scanner** - Entrada de datos y validación
2. **02-archivos** - Lectura y escritura de ficheros
3. **03-arraylist** - Colecciones dinámicas
4. **04-objetos** - Programación orientada a objetos
5. **05-metodos-avanzados** - Métodos flexibles y reutilizables
6. **06-excepciones** - Manejo profesional de errores
7. **07-herencia** - Reutilización de código y polimorfismo

## 🚀 Cómo usar

Cada carpeta contiene:
- **Tema.md** - Guía teórica completa
- **EJERCICIOS_PASO_A_PASO.md** - Ejercicios con pasos
- **PLANTILLAS/** - Archivos para resolver (con TODO comments)
- **SOLUCIONES/** - Soluciones completas
- **README.md** - Explicación específica del tema

## 📝 Flujo recomendado

Para cada tema:

1. **Lee la teoría**
   ```
   Abre: 01-scanner/Scanner.md
   ```

2. **Entiende los ejercicios**
   ```
   Abre: 01-scanner/EJERCICIOS_PASO_A_PASO.md
   ```

3. **Resuelve en VSCode**
   ```
   Abre: 01-scanner/PLANTILLAS/
   Sigue los TODO comments
   ```

4. **Compara tu solución**
   ```
   Abre: 01-scanner/SOLUCIONES/
   Revisa cómo se resolvió
   ```
