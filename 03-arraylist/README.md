## ✨ Conceptos Clave

```java
// Crear
ArrayList<Integer> numeros = new ArrayList<>();

// Operaciones
numeros.add(5);
numeros.get(0);
numeros.remove(0);
numeros.size();

// Recorrer
for (int n : numeros) { }

// Buscar
int index = numeros.indexOf(5);

// Ordenar
Collections.sort(numeros);
```
